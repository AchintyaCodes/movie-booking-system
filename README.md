CineVault is a premium, **Netflix-style** Movie Booking System. This version features a stunning glassmorphic UI, real-time seat selection, and a smooth cinematic intro.

## ✨ Features

- **📺 Netflix Intro**: Immersive brand animation on startup.
- **💎 Glassmorphic UI**: High-end aesthetic with blurred overlays and deep blacks.
- **💺 Visual Seat Selection**: Interactive grid to pick your favorite seats.
- **🔐 Secure Auth**: Built-in authentication using Bcrypt.
- **📂 Legacy Support**: The original Node.js version is preserved in the `legacy/` directory.

## 🛠️ Tech Stack

- **Frontend/Backend**: Python & Streamlit
- **Database**: SQLite (Ported for portability)
- **Security**: Bcrypt Hashing

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the App**:
   ```bash
   streamlit run streamlit_app.py
   ```

3. **Admin Settings**:
   The admin panel and database logic are pre-seeded with movies and shows.
